# 常见问题

- 为什么图片左上角有一小块类似马赛克区域？
  > 在例如 iqoo13 等机型上可见，可以尝试升级系统解决，原因尚不明确

- 为什么重新安装应用后扫描的本地歌曲信息不是之前我编辑后的？
  > 建议安装新版本时不要卸载旧版本

- 为什么有些本地歌曲没有时长信息？
  > 从 bilibili 下载的歌曲没有时长信息，你可以编辑信息进行更正

- 为什么搜索出来的合集不能自动播放下一曲？
  > 因为没有后端，不知道下一曲是哪首，虽然可以临时建个表，但是比较多余，我们有待播放列表，所以你可以把想听的音频添加到待播放列表
